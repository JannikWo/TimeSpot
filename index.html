Capture places over time

Capture the same place again and again — and see how it changes over time. Create visual timelines from your photos.

What this app does

  • Save locations as "spots"
  • Take photos from the same perspective
  • Compare images over time

Privacy Policy (DSGVO)

Datenschutzerklärung

Der Schutz deiner Daten ist uns wichtig. Diese App verarbeitet personenbezogene Daten gemäß der Datenschutz-Grundverordnung (DSGVO).

Welche Daten werden verarbeitet?

  • Account-Daten (z. B. E-Mail-Adresse)
  • Fotos, die du hochlädst oder aufnimmst
  • Spot-Daten (z. B. Standortinformationen, Metadaten)

Öffentliche und private Inhalte

Öffentliche Spots werden über unseren Server bei Supabase (Europa) gespeichert und können von anderen Nutzern eingesehen werden.

Private Spots werden ausschließlich lokal auf deinem Gerät gespeichert und nicht an externe Server übertragen.

Server & Speicherung

Für öffentliche Inhalte nutzen wir einen Dienst von Supabase mit Serverstandort in Europa. Die Datenverarbeitung erfolgt innerhalb der EU.
